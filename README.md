# Project-website
The proposed Waste Management System is a web application developed to manage
waste pick-up and disposal activities in an organized and efficient manner. The system
provides a digital platform that connects users and scarp dealers to streamline waste
handling processes and reduce manual coordination. The primary objective of this project
is to reduce improper waste disposal and minimize manual coordination by automating
waste management operations.
The system is strategically divided into two integrated modules: User and Scrap Dealers.
The User module allows individuals to register, manage their profiles and identify waste
using an AI-based identifier, submit scrap pickup request, view nearby disposal centers
and interact with the system through a Chatbot for queries and assistance. These
features enable users to manage waste related activities in a simple and user-friendly
way.

The Scrap dealer’s module is designed to support service providers in managing pickup operations effectively. Scrap dealers can register on the system, manage services by
adding pick-up requests, accept and reject user requests, handle user management and
process scrap value payments .This module helps scrap dealers organize their operations
and respond to user requests efficiently.
Overall, the Waste Management System streamlines waste handling operations,
improves user experience and supports a cleaner and more organized environment. By
reducing manual work and enhancing automation, the project provides an effective
solution for modern waste management needs
